# Water Quality Dashboard (SDG 6)


## Overview
Interactive dashboard to monitor water quality indicators:
- pH, Hardness, Solids, Chloramines, etc.
- Potability counts
- Interactive plots


## Folder Structure
- `dashboard/` : Streamlit app (`app.py`)
- `notebooks/` : Data cleaning and EDA
- `requirements.txt` : Python packages


## How to Run
1. Install dependencies:
pip install pandas plotly streamlit
2. Launch the dashboard:
streamlit run dashboard/app.py

## Dataset
Source: [Kaggle Water Potability Dataset](https://www.kaggle.com/adityakadiwal/water-potability)
